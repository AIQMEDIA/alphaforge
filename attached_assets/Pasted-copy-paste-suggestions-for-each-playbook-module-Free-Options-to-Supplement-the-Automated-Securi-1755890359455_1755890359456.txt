🚀 Free Options to Supplement the Automated Security Playbook
1. Logging & Forensic Storage
Use local JSON, CSV, or SQLite files for log storage (as per the schema above).

Free database options include SQLite (Python built-in), TinyDB (pure Python, no server), or Google Sheets API (limits apply).

Example: SQLite for Logging

python
import sqlite3
conn = sqlite3.connect('activity_log.db')
c = conn.cursor()
c.execute('CREATE TABLE IF NOT EXISTS events (timestamp TEXT, event_type TEXT, details TEXT)')
def log_event(event_type, details):
    import json, datetime
    c.execute('INSERT INTO events VALUES (?, ?, ?)', (datetime.datetime.utcnow().isoformat(), event_type, json.dumps(details)))
    conn.commit()
2. Alerts & Notifications
Email: Use Gmail SMTP (with an app password) or Mailgun’s free tier.

Slack/Discord: Free workspaces/channels; use incoming webhooks for instant team alerts.

Telegram bots: Free to set up, great for instant notifications; see Python Telegram Bot.

4. Session Control/Security Measures
Use in-memory dicts or persist to SQLite/JSON files for session rate limits and blocklists.

For geographic/device fingerprinting: Libraries like user-agents (free) or GeoLite2.

Example: Simple IP Rate Limit

python
from collections import defaultdict
import time
ip_hits = defaultdict(list)
def rate_limit(ip):
    now = time.time()
    ip_hits[ip] = [t for t in ip_hits[ip] if now - t < 60]
    ip_hits[ip].append(now)
    return len(ip_hits[ip]) > 10  # block if >10 requests/min
